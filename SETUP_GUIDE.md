# Setup Guide

## Prerequisites

### Required Software
- **Java 21** or later
- **Maven 3.6** or later
- **Git** (for cloning repository)

### Verify Installation
```bash
# Check Java version
java -version
# Should show: java version "21" or higher

# Check Maven version  
mvn -version
# Should show: Apache Maven 3.6.x or higher

# Check Git version
git --version
